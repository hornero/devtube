# DevTube

Catalogo de videos de Youtube referentes al desarrollo de aplicaciones.

## Instalación

Clonar el repositorio.
```
git clone https://github.com/hornero/devtube.git
```

Instalar dependencias.
```
npm install
```

## Ejecución

Iniciar el servidor web.
```
npm start
```
